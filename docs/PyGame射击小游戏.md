```
已实现的功能
🎮 核心玩法
- 鼠标瞄准：炮台跟随鼠标方向
- 空格射击：发射实体炮弹
- 自动声呐：每4秒自动向鼠标方向发射扇形声呐波
- 充能环：炮台下方显示环形充能进度条
🎯 目标系统
- 量子隐形：未显形目标不可击中
- 受惊反应：被声呐扫到后0.5秒内加速并随机偏转
- 方向箭头：显形目标显示移动方向
- 碰墙反弹：目标匀速直线运动，碰墙反弹
🎁 道具系统（3种）
1. *全频扫描(W)*：下一次声呐变为360°环形扫描，显形4秒
2. *黏着信标(S)*：下一次炮弹命中后标记目标6秒
3. *跳弹散芯(R)*：下一次炮弹反弹后分裂为3发弹片
📊 计分系统
- 常规击杀：+100分
- 盲狙击杀：+400分（未显形时命中）
- 窗口连击：+50×N（单次显形期内多杀）
- 极速反应：+50分（显形后1秒内击杀）
- 噪音惩罚：连续3发未命中，2秒充能暂停
📈 难度曲线
- 初始2个目标，每800分增加1个目标
- 目标速度+8%
- 4个目标后出现静默目标（只显示闪烁红点）
✨ 视觉效果
- 发光效果、轨迹拖尾、死亡动画
- 网格背景、半透明效果
- 击杀信息浮动显示
```
